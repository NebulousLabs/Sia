Public Beta
Today we are pleased to announce Sia's second public beta release, v0.3.0. Download links are listed at the bottom of the page (64-bit only).

Sia is a new, highly efficient decentralized file storage network that operates using cryptocurrency-based compensation. Hosts can join Sia, offering their local storage space to consumers in return for compensation. Consumers can then rent storage from these hosts by spending cryptocurrency. Read more â†’

Please bear in mind that Sia is undergoing rapid development. Certain aspects of the software may be unavailable or buggy. In addition, network actions such as uploading files and transferring money may take a long time to propagate. These delays will not be present in the 1.0 release.

We would like to thank all of our beta testers, whether past, present, or future. Sia is an open source project, and your contributions are invaluable to us. We encourage testers to visit Sia's GitHub page and make bug reports, feature requests, or pull requests.

Over the next few months, we'll be pushing out incremental updates to Sia. Our next major release is set for April 2015. Stay up to date by following Nebulous on Twitter or Facebook.
